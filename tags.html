Common HTML Tags

HTML tags are the building blocks of a web page. Here are some of the most commonly used HTML tags:

  • <h1> to <h6>: Used for headings. <h1> is the highest level, and <h6> is the lowest.
  • <p>: Defines a paragraph.
  • <a>: Defines a hyperlink.
  • <img>: Embeds an image.
  • <ul> and <ol>: Define unordered and ordered lists, respectively.
  • <li>: Defines a list item.

Each tag serves a unique purpose, and learning how to use them effectively will allow you to create structured web pages.
