Basic HTML Tags

Here are some of the most commonly used HTML tags:

  • <h1> to <h6>: Header tags used for titles.
  • <p>: Paragraph tag for text.
  • <a>: Anchor tag for hyperlinks.
  • <img>: Image tag for embedding images.
  • <ul> and <ol>: Unordered (bulleted) and ordered (numbered) list tags.

Example:

            
<p>This is a paragraph</p>
<a href="https://example.com">Click here</a>
