Common HTML Tags

HTML tags are used to create elements on a web page. Below are some common HTML tags:

  • <div>: Defines a division or section.
  • <p>: Defines a paragraph.
  • <a>: Defines a hyperlink.
  • <h1> to <h6>: Define HTML headings.
  • <img>: Defines an image.
  • <ul>: Defines an unordered list.
  • <ol>: Defines an ordered list.

Each tag has specific attributes that help define the content's behavior and appearance on the page.
