The Structure of an HTML Document

HTML documents follow a basic structure that is crucial for proper rendering in web browsers. Here is a simple example of an HTML structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is a simple HTML structure.</p>
  </body>
</html>

In the example above, we have the following key elements:

  • DOCTYPE: Specifies the document type (HTML5 in this case).
  • <html>: Root element that contains all HTML content.
  • <head>: Contains metadata like title and meta tags.
  • <body>: Contains the actual content of the page that will be displayed.
