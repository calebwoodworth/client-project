Basic HTML Document Structure

An HTML document has a basic structure that includes the following:

  • DOCTYPE Declaration: Tells the browser the version of HTML.
  • HTML Tag: Contains the whole HTML document.
  • Head Tag: Contains meta information, title, and links to CSS/JS files.
  • Body Tag: Contains the visible content of the webpage.
